Funhouse!

This is a page to show off the capabilities of my scaffold.css It's pretty much just a bunch of elements (such as forms and lists). It can be used as some form of manual regression testing, if I ever make any changes to the baseline styles.

It borrows heavily from Skeleton and html5boilerplate.com. If you are looking for a CSS framework or starting point, I suggest you first look at one of those projects. They are much better supported.

You can have a look through the code, and fork it on GitHub
